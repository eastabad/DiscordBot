### 2. 测试格式
在report频道中发送以下任意格式的消息：

**基础格式：**
```
AAPL 15m
TSLA 1h
NVDA 4h
```

**中文格式：**
```
AAPL 15分钟
TSLA 1小时
NVDA 4小时
```

**支持的时间框架：**
- `15m`, `15分钟`, `15min`
- `1h`, `1小时`, `1hour`, `60m`
- `4h`, `4小时`, `4hour`, `240m`

1. **机器人检测：** 机器人将自动检测到您的请求

## 🔧 故障排除

### 如果没有收到报告：

1. **检查频道名称：** 确保频道名包含"report"
2. **检查格式：** 确保使用正确的"股票代码 时间框架"格式
3. **检查用户限制：** 每日限制3次请求
4. **检查TradingView数据：** 需要有相应的历史数据

- 说明TradingView暂无该股票的详细技术指标数据
